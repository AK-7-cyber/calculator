Newton–Raphson Method
Find roots of f(x)=0 using the tangent line. Quadratic convergence — decimal places roughly double each iteration.

Lagrange Interpolation
Estimate f(x) at any point from known data points. Constructs a unique polynomial through all given data.

Basis Polynomial
Lᵢ(x) = 1 at xᵢ and 0 at all other xⱼ — unique interpolation guaranteed.
Uniqueness
For n+1 distinct points, exactly one polynomial of degree ≤ n passes through all of them.
Caution
Runge's phenomenon: many equally-spaced points can cause oscillation near edges.

Euler's Method

Algorithm
Start at (x₀,y₀) → slope k=f(xₙ,yₙ) → step: yₙ₊₁=yₙ+hk, xₙ₊₁=xₙ+h
Error
First-order method. Global error O(h). Halving h roughly halves the error.
Stability
Conditionally stable. For stiff problems use smaller h or implicit Runge-Kutta methods.
